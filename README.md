## 🛠️ Technical Stack & Architecture
- **Semantic HTML5:** Clean DOM structure using `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, and `<footer>` tags to completely eliminate unnecessary `div` nesting.
- **CSS3 Layouts:** Complex responsive grid architecture using a combination of **12-Column CSS Grid** for components and **Flexbox** for individual structural alignment (Header/Navbar components).
- **BEM Methodology:** Clean and scalable CSS class naming convention using the Block-Element-Modifier structural pattern.
